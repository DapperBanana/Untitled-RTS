Displaying the camera frustum on the minimap provides crucial feedback to the player. They can immediately understand which portion of the game world is currently visible, aiding in navigation and strategic awareness. This is a standard RTS UI pattern.  Implementing this early improves the overall user experience. I'd be interested to see how it's implemented -- probably a textured rectangle that mirrors the main camera's aspect ratio and rotation.
